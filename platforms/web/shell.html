Kirikiroid2 Web — Browser-based KiriKiri2 Engine

Run KiriKiri2 (吉里吉里2) visual novels directly in your browser powered by WebAssembly. Load .xp3 game archives or game folders — no installation required. Supports game save/load persistence, WebGL hardware-accelerated rendering, and works on both desktop and mobile devices.

Features

  • Load .xp3 game files via drag & drop, file picker, or URL
  • Extract and run games from ZIP archives
  • Persistent save data with IndexedDB save spaces
  • File System Access API for direct directory access
  • WebGL rendering with mobile shader compatibility
  • Full KiriKiri2 TJS2 scripting engine support
